Multiplying matrices and vectors

Matrix-vector product

To define multiplication between a matrix AA and a vector xx (i.e., the matrix-vector product), we need to view the vector as a column matrix. We define the matrix-vector product only for the case when the number of columns in AA equals the number of rows in xx. So, if AA is an m×nm×n matrix (i.e., with nn columns), then the product AxAx is defined for n×1n×1 column vectors xx. If we let Ax=bAx=b, then bb is an m×1m×1 column vector. In other words, the number of rows in AA (which can be anything) determines the number of rows in the product bb.

The general formula for a matrix-vector product is

Ax=⎡⎣⎢⎢⎢⎢⎢a11a21⋮am1a12a22⋮am2……⋱…a1na2n⋮amn⎤⎦⎥⎥⎥⎥⎥⎡⎣⎢⎢⎢⎢x1x2⋮xn⎤⎦⎥⎥⎥⎥=⎡⎣⎢⎢⎢⎢a11x1+a12x2+⋯+a1nxna21x1+a22x2+⋯+a2nxn⋮am1x1+am2x2+⋯+amnxn⎤⎦⎥⎥⎥⎥.Ax=[a11a12…a1na21a22…a2n⋮⋮⋱⋮am1am2…amn][x1x2⋮xn]=[a11x1+a12x2+⋯+a1nxna21x1+a22x2+⋯+a2nxn⋮am1x1+am2x2+⋯+amnxn].
Although it may look confusing at first, the process of matrix-vector multiplication is actually quite simple. One takes the dot product of xx with each of the rows of AA. (This is why the number of columns in AA has to equal the number of components in xx.) The first component of the matrix-vector product is the dot product of xx with the first row of AA, etc. In fact, if AA has only one row, the matrix-vector product is really a dot product in disguise.

For example, if

A=[10−1−321]A=[1−120−31]
and x=(2,1,0)x=(2,1,0), then
Ax=[10−1−321]⎡⎣⎢210⎤⎦⎥=[2⋅1−1⋅1+0⋅22⋅0−1⋅3+0⋅1]=[1−3].Ax=[1−120−31][210]=[2⋅1−1⋅1+0⋅22⋅0−1⋅3+0⋅1]=[1−3].

Matrix-matrix product

Since we view vectors as column matrices, the matrix-vector product is simply a special case of the matrix-matrix product (i.e., a product between two matrices). Just like for the matrix-vector product, the product ABAB between matrices AA and BB is defined only if the number of columns in AA equals the number of rows in BB. In math terms, we say we can multiply an m×nm×n matrix AA by an n×pn×p matrix BB. (If pp happened to be 1, then BB would be an n×1n×1 column vector and we'd be back to the matrix-vector product.)

The product ABAB is an m×pm×p matrix which we'll call CC, i.e., AB=CAB=C. To calculate the product BB, we view BB as a bunch of n×1n×1 column vectors lined up next to each other:

⎡⎣⎢⎢⎢⎢⎢b11b21⋮bn1b12b22⋮bn2……⋮…b1pb2p⋮bnp⎤⎦⎥⎥⎥⎥⎥=⎡⎣⎢⎢⎢⎢⎢⎡⎣⎢⎢⎢⎢b11b21⋮bn1⎤⎦⎥⎥⎥⎥⎡⎣⎢⎢⎢⎢b12b22⋮bn2⎤⎦⎥⎥⎥⎥⋯⎡⎣⎢⎢⎢⎢⎢b1pb2p⋮bnp⎤⎦⎥⎥⎥⎥⎥⎤⎦⎥⎥⎥⎥⎥[b11b12…b1pb21b22…b2p⋮⋮⋮⋮bn1bn2…bnp]=[[b11b21⋮bn1][b12b22⋮bn2]⋯[b1pb2p⋮bnp]]
Then each column of CC is the matrix-vector product of AA with the respective column of BB. In other words, the component in the iith row and jjth column of CC is the dot product between the iith row of AA and the jjth column of BB. In math, we write this component of CC as cij=ai1b1j+ai2b2j+⋯+ainbnjcij=ai1b1j+ai2b2j+⋯+ainbnj.

An example help makes the process clear. Let AA be the 2×32×3 matrix

A=[0−44−3−20]A=[04−2−4−30]
and BB be the 3×23×2 matrix
B=⎡⎣⎢0121−13⎤⎦⎥.B=[011−123].
Then,
AB=[0−44−3−20]⎡⎣⎢0121−13⎤⎦⎥=[0⋅0+4⋅1−2⋅2−4⋅0−3⋅1+0⋅20⋅1+4⋅(−1)−2⋅3−4⋅1−3⋅(−1)+0⋅3]=[0+4−40−3+00−4−6−4+3+0]=[0−3−10−1].AB=[04−2−4−30][011−123]=[0⋅0+4⋅1−2⋅20⋅1+4⋅(−1)−2⋅3−4⋅0−3⋅1+0⋅2−4⋅1−3⋅(−1)+0⋅3]=[0+4−40−4−60−3+0−4+3+0]=[0−10−3−1].
